# Budgety-JS

Small JavaScript application to manage budget income and expenses

## Getting Started



### Prerequisites

Install software:

```
git bash
```
### Installing

1. Clone the repo

```
git clone https://github.com/HarishLanke/Budgety-JS.git
```

2. Open Project Folder

3. Open index.html in your favorite browser

4. Explore

```
Enjoy :)
```
